todo dev tem seu setup. aqui vai o meu, simples e funcional.

## editor

VS Code com alguns plugins essenciais:

- **JetBrains Mono** como fonte (a ligadura é linda)
- Tema escuro padrão com ciano como accent
- `rust-analyzer`, `ESLint`, `Tailwind CSS IntelliSense`

## terminal

Windows Terminal + PowerShell. config minimalista, sem prompt enfeitado.

## workflow

1. planejo rápido no papel ou no bloco de notas
2. código no editor com preview automático
3. testo no navegador
4. commit pequeno e frequente

> "ferramenta boa é a que não atrapalha"

não troco de setup todo mês. quando acho algo que funciona, fico com isso até doer.
